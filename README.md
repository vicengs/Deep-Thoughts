## Description
  
Application to allow user add expenses and deposits to their budget with or without a connection. If the user enters transactions offline, the total should be updated when they're brought back online.

## Built with

- JavaScript
- Node
- Express
- Service Workers
- Manifest
- indexedDB
- MongoDB
- Mongoose
